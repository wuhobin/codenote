# Vibe coding

> 用自然语言描述意图，让 AI 把代码写出来。

Vibe coding 是 AI 时代的新型编程方式——你不需要逐行敲代码，而是用自然语言描述你想要什么，AI 帮你生成代码。你负责「想清楚要什么」，AI 负责「怎么实现」。

## 核心理念

- **意图驱动**：关注「做什么」而非「怎么做」
- **快速迭代**：想法到原型的周期从天级缩短到分钟级
- **技能杠杆**：一个开发者可以完成过去一个团队的工作量
- **持续学习**：与 AI 对话的过程就是学习新技术的最佳方式
